# Kinfolk

**Your family's digital gathering place.**

Kinfolk is an open-source, privacy-first smart display for families. A 24/7 dashboard that brings your family together through shared calendars, voice commands, photo memories, and everyday coordination.

## What is Kinfolk?

Think Amazon Echo Show meets Magic Mirror, but:
- ✅ Fully open-source
- ✅ Privacy-first (local processing)
- ✅ Family-focused features
- ✅ Beautiful, customizable UI
- ✅ Voice-controlled
- ✅ No subscriptions, ever

## Core Features

- 🕐 Always-on family dashboard
- 🎙️ Voice assistant (wake word + commands)
- 📅 Shared calendar & to-do lists
- 🌤️ Weather & news
- 📸 Photo frame mode
- 🎵 Music player
- 👨‍👩‍👧 Multi-user profiles
- 🏠 Smart home control
- 🔐 Privacy-first design

## Hardware

Designed for:
- **Display:** 1080x1920 vertical monitor
- **Computer:** Raspberry Pi 5 or Mini PC
- **Input:** Touchscreen + USB microphone
- **Optional:** Camera (face recognition, video calls)

## Tech Stack

- **Frontend:** Flutter Desktop
- **Voice:** Rhasspy + Whisper API
- **Backend:** Home Assistant + Custom APIs
- **Database:** SQLite + optional Supabase sync
